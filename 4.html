4. HTML이란?

한글: HTML (Hypertext Markup Language)은 웹 페이지와 그 내용을 구조화하기 위해 사용하는 코드이다.
예를 들면, 콘텐츠는 여러 개의 문단, 글 머리 목록이 구조화된 것이거나 사진이나 데이터 테이블일 수도 있다.
제목과 같이 이 글은 HTML과 그 기능에 대한 기본적인 이해를 제공할 것이다.

English: Hypertext Markup Language (HTML) is the standard markup Language for creating web pages and web applications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.
